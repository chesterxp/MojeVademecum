Co daje Use strict


        1.	Usunięcie właściwości WITH
        2.	nie można wyświetlić wartości zmiennej jesli nie jest ona zaadeklarowana:
            a.	zmienna = "Samochód"; console.log(zmienna) //ReferenceError: zmienna is not defined
        3.	ograniczenie możliwości nadpisania zmiennych globalnych
            a.	NaN = 3; // TypeError: Cannot assign to read only property 'NaN' of object '#'
            b.	 Infinity = 3; // TypeError:
        4.	ograniczenie możliwości zadeklarowania tej samej zmiennej
            var test = {
                    a: 1,
                    b: 2,
                    a: 3 // error
                };
        5.	ograniczenie możliwości zadeklarowania tej samej zmiennej w funkcji
        a.	function overrideParameters(a, b, a) { // SyntaxError: Duplicate parameter name not allowed in this context
        6.	Nie można również nadpisywać zmiennej „arguments”. Taka próba prowadzi do błędu
            function argumentsExample1(a, b, c) {
                "use strict";
                arguments = 1; //  SyntaxError: Unexpected eval or arguments in strict mode
            }
        7.	nie można nadpisać zmiennej "arguments:
            argumentsExample2(1, 2, 3);
            function argumentsExample2(a, b, c) {
                "use strict";
                a = -1;
                console.log(arguments); // 1, 2, 3
            }
        8.	Tryb ścisły nie pozwala używać słów, które będą (lub już są) uznane za słowa kluczowe standardu. Do tych słów należą:
            a.	implements
            b.	interface
            c.	let
            d.	package
            e.	private
            f.	protected
            g.	public
            h.	static
            i.	yield
        9.	kasowanie zmiennej lub funkcji za pomocą „delete”
        10.	używanie składni ósemkowej (var x = 010)
        11.	nadpisywanie właściwości tylko do odczytu
        12.	użycie zmiennej „eval” jako zmiennej (var eval = „test”)
        13.	Na koniec warto podać jakie wersje wspierają tryb ścisły:
            a.	IE od wersji 10.
            b.	Firefox od wersji 4.
            c.	Chrome od wersji 13.
            d.	Safari od wersji 5.1.
            e.	Opera od wersji 12.

ES6

            1.	Wykorzystanie let, const. Działają tylko wewnątrz bloku a nie poza nim.
            2.	template string. Mozliwośc zapisywania ciągów znaków w róznych liniach za pomocą: ``
            3.	dodstanie sie do elementów tablicy za pomoca skróconego zapisu ...arr(3 kropki i nazwa tablicy)
            4.	wypisanie pozostałych argumentów z funkcji:
                    function argumenty(a1,a2, ...rest){//1,2,3,4,5,6,7
                        console.log('pierwsze dwa to: ',a1, a2);//1,2
                        console.log('pozostałe argumenty to: ', ...rest);//3,4,5,6,7
                    }
            5.	Nowa forma zapisywania funkcji 
                    a.	old -> function wypisz(wyraz){console.log('Twój wyraz to:', wyraz);}
                    b.	new -> var wypisz2 = (wyraz) => {console.log('Twój wyraz to:', wyraz);}
            6.	Możliwośc przypisywania wartości domyślnej w funkcji
                    function def(parm = "Auto"){
                        console.log(parm);
                    }
                    def()//auto
                    def('dom');//dom

ScrollBar

            //szerokość i wysokość scrolbarów
            body::-webkit-scrollbar {
                width: 13px;
                height: 13px;
            }

            //kolor paska
            body::-webkit-scrollbar-track {
                background-color: #2F3336;
                -webkit-box-shadow: inset 0 0 5px rgba(0, 0, 0, 0.8);
            }

            //kolor suwaka
            body::-webkit-scrollbar-thumb {
                background-color: #1c74b2; // border-radius: 20px;
                -webkit-box-shadow: inset 0 0 5px rgba(0, 0, 0, 0.8);
            }

            //kolor wypełnienia cornera
            body::-webkit-scrollbar-corner {
                background: #2F3336 url('../scroll/corner.png') 2px 2px no-repeat;
            }

            //strzałki góra dół lewa prawa
            body::-webkit-scrollbar-button:vertical:decrement {
                background: #2F3336 url(../scroll/up.png) 2px 2px no-repeat;
                -webkit-box-shadow: inset 0 0 5px rgba(0, 0, 0, 0.8);
            }

            body::-webkit-scrollbar-button:vertical:increment {
                background: #2F3336 url(../scroll/down.png) 2px 3px no-repeat;
                -webkit-box-shadow: inset 0 0 5px rgba(0, 0, 0, 0.8);
            }

            body::-webkit-scrollbar-button:horizontal:decrement {
                background: #2F3336 url(../scroll/left.png) 1px 2px no-repeat;
                -webkit-box-shadow: inset 0 0 5px rgba(0, 0, 0, 0.8);
            }

            body::-webkit-scrollbar-button:horizontal:increment {
                background: #2F3336 url(../scroll/right.png) 3px 2px no-repeat;
                -webkit-box-shadow: inset 0 0 5px rgba(0, 0, 0, 0.8);
            }
